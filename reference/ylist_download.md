# Download the public YList tab-delimited data file

Downloads the public YList tab-delimited data file into the user's R
cache. The file is not bundled with the package.

## Usage

``` r
ylist_download(overwrite = FALSE)
```

## Arguments

- overwrite:

  Logical. If `FALSE`, an existing cached file is reused.

## Value

The path to the cached file, invisibly.
